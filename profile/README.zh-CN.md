## 🌏 关于我们

**星辰实验室（Stars Lab）** 是一家独立的研究与开发实验室，专注于探索三大领域交汇处的**下一代技术**：

| | 方向 | 我们在做什么 |
|---|---|---|
| ⛓️ | **区块链基础设施** | 高性能、安全、模块化的区块链协议 |
| 🤖 | **人工智能** | 面向自主性、推理与智能系统的应用 AI |
| 🦾 | **人形机器人** | 机器人技术、控制系统与真实世界部署 |

我们的使命是不断突破**去中心化、智能化与人机协作**的边界——构建开放、可扩展、面向未来的系统。
