## Repository Purpose

This is the special `.github` repository for the **stars-labs** GitHub organization (Stars Lab — an R&D lab focused on blockchain infrastructure, AI, and humanoid robotics). It contains no application code, build system, or tests.

- `profile/README.md` — the organization's public profile page, rendered on https://github.com/stars-labs. This is the main file edited in this repo.
- `README.md` — minimal repo README pointing to https://stars-labs.github.io/stars-labs/

## Working in This Repo

- Changes are Markdown-only; there is nothing to build, lint, or test.
- Edits to `profile/README.md` are immediately public on the organization's GitHub page once pushed to `main` — review wording carefully before committing.
- Keep the existing tone and emoji-heavy style of the profile README when editing it.
- `profile/README.md` is bilingual (English + 简体中文, "Stars Lab · 星辰实验室"). Any content change must be applied to both language sections to keep them in sync.
